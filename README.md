Android Registration Form App
📱

Deskripsi Project

Project ini merupakan aplikasi Android sederhana yang dibuat menggunakan Android Studio untuk memenuhi tugas E-Learning pertemuan 5.
Aplikasi ini menampilkan form registrasi lengkap dengan berbagai komponen UI dan validasi input yang umum digunakan dalam pengembangan aplikasi Android modern.

Project ini mengimplementasikan beberapa konsep penting seperti:

-Form input dengan Material Design

-Validasi data pengguna

-Selection controls

-Spinner dan dialog

-Publikasi project ke GitHub
